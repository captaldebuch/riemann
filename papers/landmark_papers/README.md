# Landmark Papers of the Riemann Hypothesis

This folder is intended to store the 100 most influential papers on the Riemann Hypothesis.

Currently, we are attempting to download:
- Riemann (1859) - "Ueber die Anzahl der Primzahlen..."
- Hadamard (1893)
- Hardy & Littlewood (1921)
- Levinson (1974)
- Conrey (1989)
- Deligne (1974)
- Montgomery (1973)
- Odlyzko (1987)
- Katz & Sarnak (1999)
- Connes (1999)
- Robin (1984)
- Lagarias (2002)

**Note:** Many of these foundational papers (like Riemann 1859, Hardy 1914, Deligne 1974) are behind publisher paywalls (Springer, AMS, etc.) or exist only as historical scans, making them impossible to automatically download via OpenAlex's open access PDF fetcher. We have logged their OpenAlex Work IDs (`W...`) but manual retrieval is required for closed-access journals.
